# Boss ops portal ÔÇö public domain + unified login

One secure HTTPS entry for **Boss UI**, **OpenClaw Control UI**, and optional **Grafana** ÔÇö without exposing Postgres, Prometheus, or raw gateway ports.

**Hermes has no separate web dashboard on Clawsum.** Long-job Hermes runs through Paperclip Ôćĺ OpenClaw (`paperclip:hermes` session). Monitor and assign Hermes work in **Boss UI** (filter assignee **Clawsum Hermes**).

Optional future: Nous **Hermes Agent dashboard** (`hermes dashboard`, :9119) ÔÇö not installed; see [HERMES-POLICY.md](./HERMES-POLICY.md) and [MASTER-TASK-LIST.md](./MASTER-TASK-LIST.md).

## Recommended approach (two layers)

### Layer 1 ÔÇö Traefik login wall (same for all hosts)

Pick one:

| Method | Best for | One login? |
|--------|----------|------------|
| **Traefik basic auth** | Fastest; Boss-only team | Same password per subdomain (one prompt each) |
| **Authelia** | Email + 2FA; production | Yes ÔÇö session cookie across subdomains |
| **Cloudflare Access** | No VPS auth service | Yes ÔÇö at Cloudflare edge |

Use `deploy/scripts/setup-ops-portal-traefik.sh` for **basic auth** (Tier 1).

### Layer 2 ÔÇö App auth (inside the wall)

| App | What Boss still does |
|-----|----------------------|
| **Boss UI** | Sign in with Paperclip user (Better Auth). Set `PAPERCLIP_PUBLIC_URL=https://bossÔÇŽ` |
| **OpenClaw** | With **trusted-proxy**: no gateway token / no device pairing. Without it: paste `OPENCLAW_GATEWAY_TOKEN` once |
| **Grafana** | `admin` + `GRAFANA_ADMIN_PASSWORD` unless you enable Grafana auth proxy |

## Quick start (basic auth + Boss UI public)

### 1. DNS

| Type | Name | Value |
|------|------|--------|
| A | `boss` | VPS public IP |
| A | `clawsum` | VPS public IP |
| A | `grafana` | VPS public IP (optional) |

### 2. `.env` on VPS

```env
TRAEFIK_HOST=yourdomain.com
BOSS_UI_HOST=boss.yourdomain.com
OPENCLAW_UI_HOST=clawsum.yourdomain.com
GRAFANA_UI_HOST=grafana.yourdomain.com
PAPERCLIP_PUBLIC_URL=https://boss.yourdomain.com

# Ops portal Traefik basic auth (generated by setup script if empty)
BOSS_OPS_AUTH_USER=boss
BOSS_OPS_AUTH_PASSWORD=change_me_strong
```

### 3. Run setup script (VPS)

```bash
cd /docker/clawsum
BOSS_HOST=boss.yourdomain.com \
OPENCLAW_HOST=clawsum.yourdomain.com \
GRAFANA_HOST=grafana.yourdomain.com \
bash scripts/setup-ops-portal-traefik.sh
```

This writes `/docker/traefik/dynamic/ops-portal.yml`, enables file provider, restarts Traefik, and restarts Paperclip with the public URL.

### 4. OpenClaw Control UI behind the same wall (trusted-proxy)

After Traefik auth works, drop gateway token prompts:

```bash
python3 /docker/clawsum/scripts/patch-control-ui-trusted-proxy.py
cd /docker/clawsum && docker compose restart openclaw-gateway
```

Requires OpenClaw **2026.6.10+** with `gateway.auth.mode: trusted-proxy`. Traefik must send `X-Forwarded-User` (the setup script configures a forward-auth shim for basic auth).

### 5. Verify

```bash
curl -u boss:PASSWORD -sS -o /dev/null -w '%{http_code}\n' https://boss.yourdomain.com/api/health
curl -u boss:PASSWORD -sS -o /dev/null -w '%{http_code}\n' https://clawsum.yourdomain.com/healthz
```

Browser: open `https://boss.yourdomain.com` Ôćĺ Traefik login Ôćĺ Paperclip login Ôćĺ tasks.

Open `https://clawsum.yourdomain.com` Ôćĺ same Traefik login Ôćĺ Control UI (no pairing when trusted-proxy is active).

## Production: Authelia (single sign-on)

For one login across all subdomains:

1. Deploy [Authelia](https://www.authelia.com/) on the VPS (or use Cloudflare Access).
2. Replace `boss-ops-auth` basic auth middleware with `forwardAuth` Ôćĺ Authelia.
3. Authelia sets `Remote-User` Ôćĺ Traefik copies to `X-Forwarded-User`.
4. Run `patch-control-ui-trusted-proxy.py` (uses `x-forwarded-user`).
5. Configure Paperclip Better Auth separately (Boss UI login) ÔÇö or disable public signup and rely on Traefik + `local_trusted` if your threat model allows it.

Authelia is **not** in compose today; treat as instance overlay when you outgrow basic auth.

## Security checklist

- [ ] All admin UIs on HTTPS only (Traefik Let's Encrypt)
- [ ] Traefik login wall on **boss**, **clawsum**, **grafana** (not on MinIO/Prometheus/Postgres ports)
- [ ] `PAPERCLIP_PUBLIC_URL` matches browser URL exactly
- [ ] `OPENCLAW_GATEWAY_TOKEN` rotated; not committed to git
- [ ] OpenClaw `gateway.trustedProxies` lists only Traefik (not `0.0.0.0/0`)
- [ ] Firewall: only `:22`, `:80`, `:443` public; `:3100`, `:48166`, `:3000` stay on loopback
- [ ] SSH tunnel fallback documented for outages

## Troubleshooting

| Symptom | Fix |
|---------|-----|
| Boss UI cookie / redirect loop | `PAPERCLIP_PUBLIC_URL` must be `https://bossÔÇŽ` with no trailing slash |
| OpenClaw 401 / pairing required | Run `patch-control-ui-trusted-proxy.py`; confirm OpenClaw version; check Traefik sends `X-Forwarded-User` |
| WebSocket 1008 | Trusted-proxy + allowedOrigins; run `patch-control-ui-origins.py` |
| Double login annoyance | Upgrade to Authelia or Cloudflare Access for shared session |
| Grafana public without auth | Add `grafana` route + middleware; never publish `:3000` to `0.0.0.0` |
